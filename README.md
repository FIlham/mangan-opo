# Mangan Opo?

Mangan Opo adalah website yang berfungsi untuk men-generate rekomendasi makanan yang cocok dikonsumsi di waktu-waktu tertentu.

## Demo

Insert gif or link to demo

## Install and Run

Anda bisa download/clone repository ini dan jalankan file `index.html`. Bisa dengan live-server ataupun direct dengan file HTML.
Demo bisa cek di [sini](https://mangan-opo.netlify.app/)

## Features

Website ini hanya sebatas generator makanan yang datanya manual diambil dari ChatGPT (bisa lihat di `data.json`).\
File JavaScript di project ini ditulis dengan style [JSDoc](https://jsdoc.app/) agar meningkatkan pengalaman developer yang asik dengan code-completion yang dihasilkan (cek demo).
